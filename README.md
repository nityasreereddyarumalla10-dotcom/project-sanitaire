# Project Sanitairè
An interactive Python application designed for a 5-minute daily health and hygiene literacy program in government schools across India.

##  Key Features
* **Localized Database**: Instantly maps major Indian municipal areas to specific seasonal health risks.
* **Gamified Learning**: Includes an interactive trivia challenge that reinforces sanitation habits.
* **Animated Terminal**: Uses custom delays to simulate a dynamic classroom interface.

##  Repository Files
* `main.py`: The complete core application code.
